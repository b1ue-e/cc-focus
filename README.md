## cc-focus

Auto-focus your terminal when Claude Code finishes responding — browse elsewhere while you wait.

### How it works

```
Claude Code Stop hook → notify.sh → Unix socket → daemon → activate terminal
```

When Claude Code completes a response, the `Stop` hook fires `notify.sh`, which forwards the event to a lightweight daemon. The daemon checks if your terminal is already in focus — if not, it brings it to the front.

### Install

```bash
make install                          # installs to ~/.local/bin
cc-focus install                      # registers Stop hook in ~/.claude/settings.local.json
cc-focus start                        # starts the daemon
```

### Commands

| Command | Description |
|---------|-------------|
| `cc-focus start` | Start daemon in background |
| `cc-focus stop` | Stop daemon |
| `cc-focus status` | Show daemon status and config |
| `cc-focus install` | Add Stop hook to Claude Code settings |
| `cc-focus uninstall` | Remove Stop hook from Claude Code settings |

### Config

`~/.config/cc-focus/config.toml` (optional — defaults work out of the box):

```toml
[terminal]
name = "Ghostty"          # your terminal app name
activate_cmd = ""         # custom activation command (optional)

[hook]
socket_path = ""          # Unix socket path (auto-generated by default)
```

### Requirements

- **macOS:** System Events permissions for osascript
- **Linux:** `wmctrl` and `xdotool` (`apt install wmctrl xdotool`)

## cc-focus

Claude Code 响应完成时自动将终端切回前台 — 等待模型输出的同时放心浏览其他页面。

### 原理

```
Claude Code Stop hook → notify.sh → Unix socket → daemon → 激活终端
```

Claude Code 完成响应时，`Stop` hook 调用 `notify.sh`，将事件转发给后台 daemon。daemon 检测终端是否已在最前，如果不在则自动激活。

### 安装

```bash
make install                          # 安装到 ~/.local/bin
cc-focus install                      # 注册 Stop hook 到 ~/.claude/settings.local.json
cc-focus start                        # 启动 daemon
```

### 命令

| 命令 | 说明 |
|------|------|
| `cc-focus start` | 后台启动 daemon |
| `cc-focus stop` | 停止 daemon |
| `cc-focus status` | 查看运行状态和配置 |
| `cc-focus install` | 添加 Stop hook 到 Claude Code 设置 |
| `cc-focus uninstall` | 移除 Stop hook |

### 配置

`~/.config/cc-focus/config.toml`（可选，默认配置即可直接使用）：

```toml
[terminal]
name = "Ghostty"          # 终端应用名
activate_cmd = ""         # 自定义激活命令（可选）

[hook]
socket_path = ""          # Unix socket 路径（默认自动生成）
```

### 依赖

- **macOS:** 需要授予辅助功能权限以使用 osascript
- **Linux:** `wmctrl` 和 `xdotool` (`apt install wmctrl xdotool`)
